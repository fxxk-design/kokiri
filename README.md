# Buds for Vue

## 约定

### 组件属性

| 语义                 | 命名           | 取值                             |
| -------------------- | -------------- | -------------------------------- |
| 指定值（受控）       | `value`        | -                                |
| 默认值（不受控）     | `defaultValue` | -                                |
| 尺寸                 | `size`         | `'large'`、`'medium'`、`'small'` |
| 形状                 | `shape`        | -                                |
| 可清除               | `clearable`    | `true`、`false`                  |
| 可搜索               | `searchable`   | `true`、`false`                  |
| 标志（相当于 `key`） | `flag`         | -                                |
| 数据源               | `dataSource`   | -                                |
| 类名                 | `className`    | -                                |

### 组件事件

| 语义             | 命名             |
| ---------------- | ---------------- |
| 弹出层可见性变化 | `visible-change` |
